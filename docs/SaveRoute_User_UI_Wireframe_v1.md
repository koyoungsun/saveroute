# SaveRoute User UI Wireframe v1.0

> 모바일 우선 PWA (430px 기준). Tailwind CSS 기반 사용자 화면 와이어프레임.

---

## 공통 사항

- 최대 너비: 430px, 중앙 정렬
- 하단 탭바: 고정 (height 64px)
- 콘텐츠 영역: padding-bottom 64px (탭바 높이만큼 확보)
- 배경색: #F8FAFC
- 카드 radius: 16px
- Primary: #2563EB
